# Monish Shekh — Developer Portfolio

Modern, responsive portfolio built with **React + Vite + Framer Motion**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Tech Stack
- React 18 + Vite 5
- Framer Motion (animations)
- React Router v6
- EmailJS (contact form)
- Lucide React + React Icons

## ⚙️ Setup

1. **Install dependencies:** `npm install`
2. **Create `.env`** in root with your EmailJS keys:
   ```
   VITE_EMAILJS_SERVICE_ID=...
   VITE_EMAILJS_TEMPLATE_ID=...
   VITE_EMAILJS_PUBLIC_KEY=...
   ```
3. **Drop your files in `/public/`:**
   - `photo.jpg` — your profile photo
   - `resume.pdf` — your resume
   - `expensely.png`, `wordanalyzer.png`, `blogapp.png`, `portfolio.png` — project screenshots
   - `/certs/` — certificate images
   - `/gallery/` — personal/project/achievement photos

4. **Run:** `npm run dev`
5. **Build:** `npm run build`
